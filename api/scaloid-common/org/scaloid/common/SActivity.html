org.scaloid.common

SActivity

trait SActivity extends Activity with SContext with TraitActivity[SActivity] with Destroyable with Creatable with Registerable

Enriched trait for the class android.app.Activity. To enable Scaloid, inherit this trait instead of the class Activity.

SActivity has shortcut methods for each lifecycle stages. These methods shorten the method overriding, which is very frequent for activity classes.

For example, instead of this:

override def onCreate(b: Bundle) {
super.onCreate(b)
// do something for onCreate
}

The equivalent is:

onCreate {
// do something for onCreate
}

or

onCreate(doSomething())

In contrast of method overriding, this shortcut can be called multiple times from different places of your code.
